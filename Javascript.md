### 평균 구하기

##### reduce

* 배열의 각 요소에 대해 Callback을 실행해 1개의 값을 반환

```javascript
// 1. 숫자 리스트 합
[1, 2, 3, 4, 5].reduce((acc, cur) => acc + cur);
```



### 가운데 글자 가져오기

##### Math.ceil

* 올림

```javascript
Math.ceil(4) // 4
Math.ceil(4.2) // 5
Math.ceil(4.7) // 5
```
